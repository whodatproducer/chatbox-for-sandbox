# chatbox-for-sandbox

to use:
1. copy the Sounds folder into the Assets folder for your project
2. copy the Punchy Player Code folder into the Code folder for your project
3. create an empty game object in the scene and attach the ChatManager.cs component
4. create a screen object in the scene and attach the chatbox.razor component
5. drag the object you created in step 4 (which has the ChatManager.cs component) onto the chatbox.razor component of the screen
  <img width="861" height="249" alt="image" src="https://github.com/user-attachments/assets/e08305ff-34c8-4cb4-bff0-7014df82be9b" />

6. attach PunchyPlayer.cs component and PlayerChat.cs component to the player prefab


**This also includes a punch and slide ability for the player. you will have to add a slide control in the project settings for it to work, or you can just disable it
