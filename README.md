# chatbox-for-sandbox

to use:
1. copy the Sounds folder into the Assets folder for your project
2. copy the Punchy Player Code folder into the Code folder for your project
3. create a screen object in the scene and attach the chatbox.razor component
4. create an empty game object in the scene and attach the ChatManager.cs component
5. attach PunchyPlayer.cs component and PlayerChat.cs component to the player prefab


**This also includes a punch and slide ability for the player. you will have to add a slide control in the project settings for it to work, or you can just disable it
